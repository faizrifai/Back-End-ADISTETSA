Prosedure Instalasi Smart Library Automation

Instalasi Server:
1. Install Server dengan nama PERPUS-SVR (OS Windows Server 2003 atau Windows XP) beserta Patch (service pack) nya.
2. Install DataBase MS SQL Server 2000 beserta update-nya, kalau servernya Win XP gunakan MS SQL Server 2000 Personal edition
   Buat password untuk user database sa menjadi revo77
   Restore DataBase SLA_SMAN4_MALANG
5. Install IIS (webserver)
   Ubah default document menjadi Home.PHP
   Ubah Webservice Extentions menjadi Allow :  All Unknown ISAPI Extension
                                               All unknown CGI Extension
6  Salin file dalam CD folder wwwroot ke dalam server folder c:\inetpub\wwwroot                                           
7. Install PHP
   Ubah PHP.INI (dalam folder windows) : 
			Register_Globals=On
			odbc.defaultlrl = 300000 (buang tanda ; )
                       	PHP Timeout = 300
                        error_reporting=E_ALL   (buang tanda ; dan error yang lain jangan dipasang)
			post_max_size = 20M
			upload_max_filesize = 20M
8. Buat ODBC System: Driver: SQL Server, Nama: SLA_SMAN4_MALANG, server: PERPUS-SVR user: sa, password: revo77, database: SLA_SMAN4_MALANG

Instalasi Workstation (Client):
1. Double Click Setup.exe
2. Ikuti petunjuk di layar sampai Finish
   Untuk windows 98 perlu diupdate (buka folder update windows 98 --> click mdac_type.exe dan ikuti petunjuk dilayar)
3. Jalankan Smart Library Automation
4. Masuk ke menu pengaturan
5. Isi data konfigurasi yang benar
6. Smart Library Automation siap digunakan


Instalasi Workstation (Client) untuk katalog online:
1. click Pada internet explorer
2. ubah Address: http://perpus-svr
